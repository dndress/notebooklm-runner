## How to use this visual guide

Use this file only as a character reference. Do not illustrate this guide, its headings, its rules, player names, Discord IDs, labels, or notes.

The full visual description is the canonical source for each character’s appearance. Never contradict it.

The Short Character Lock is a compact reminder for panel prompts. It does not replace the full description.

When a named character is the main focus of a panel, use the full visual description to preserve detail.

When multiple named characters appear in one panel, use each character’s Short Character Lock to keep them distinct and avoid mixing traits.

When a panel names a character, do not replace that character with a generic fantasy figure or another character from the guide.

If a character has “Do not confuse with” notes, follow them strictly.

For final comic pages, include only the story narration text. Do not print visual instructions, character locks, guide notes, labels, or production notes.

## Final PDF visual style guide

Use this section only to guide the final comic PDF. Do not illustrate or print this section, its heading, its rules, labels, file names, or instruction text.

Create a dark fantasy graphic novel comic based on the session outline. This is a graphic novel, not a presentation deck. Do not add slide titles, bullet points, presenter-style layouts, decorative headings, icons, or presentation elements.

Use the session outline as the main source for page order, panel order, visual action, and narration text. Use the character visual guide as the canonical source for the appearance of every named player character. Do not invent or change a named character’s gender, ancestry, hair, clothing, weapons, magic style, or main visual anchors.

Only print the story narration inside the comic panels. Use only the text after `Narration:` from the session outline. Do not print labels such as `Visual:`, `Narration:`, `Required visible characters:`, `Short character lock:`, `Production note:`, or any other guide text.

Style: dark fantasy horror graphic novel illustration with mature comic-book aesthetics, neo-noir lighting, heavy black ink shadows, sharp readable linework, limited color palette, and high contrast. Use deep blacks, cold teal-blue shadows, muted parchment tones.

The pages should feel ominous, nocturnal, serious, adult-targeted, and story-driven. Use strong silhouettes, dramatic single focal points, cinematic cropping, screenprint-like color blocking, and controlled gritty texture. Avoid cute, comedic, anime, photorealistic, 3D-rendered, glossy, or overly clean fantasy art. no empty white space 

Use texture carefully. Add grain, rough ink, smoke, cracks, sparks, and worn paper texture only where they support the focal scene. Avoid muddy rendering, excessive scratch marks, noisy grunge overlays, over-hatching, and deep-fried texture.

Page layout: Use a full-bleed dark graphic novel page layout. The entire page should feel illustrated, atmospheric, or intentionally inked. Do not leave plain white gutters, white margins, empty white wedges, or unused presentation-like space between panels. If gutters or borders are visible, make them black, dark, textured, painted, or integrated into the page atmosphere.

Panels may be irregular, tilted, overlapping, or separated by dark ink gutters, but the page must feel like one complete comic composition. Avoid clean white slide backgrounds, floating panels on white space, or collage layouts with unused empty areas. Narration boxes may use parchment tones, but the surrounding page should remain dark and story-focused.

Every panel should prioritize story clarity. If a panel names a required visible character, that character must appear and must be visually identifiable using the visual guide. If a panel names only a subject or object, it is acceptable to show the subject without showing the heroes.
